# TEKBIZ

SaaS e-commerce mobile-first pour commerçants africains.

## Démarrage

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Pages

- `/` — Landing page
- `/login` — Connexion
- `/register` — Inscription
- `/dashboard` — Tableau de bord marchand
- `/dashboard/products` — Gestion produits
- `/dashboard/orders` — Gestion commandes
- `/dashboard/payments` — Historique paiements
- `/dashboard/store` — Paramètres boutique
- `/shop/[slug]` — Vitrine publique
